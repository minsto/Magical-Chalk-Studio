 #################################################################################################################
 /chalk : L’objet dans vos mains est le premier état.
/chalk magic setpoint : Il suffit de cliquer pour 4 points : 1 x vers le bas, 1 x vers le haut, 1 y vers le bas et 1 y vers le haut. Maintenant, vous utilisez :
/chalk magic create [nom_de_votre_structure] : Créez votre structure avec le nom choisi.
/chalk cancel : Annule le point de référence.
/magichalk list : Liste tous les noms de structures existant dans votre monde et les importe.
/magichalk import : Importe une structure depuis votre fichier de configuration "magicalchalk" dans le dossier des mods.
/magichalk export : Exporte votre structure créée dans votre monde vers la configuration "magicalchalk" pour l’envoyer à votre ami ou à d’autres personnes.
/magichalk [nom_de_structure] : Vous ajoutez l’objet "magic chalk" dans votre main, puis il suffit de toucher un bloc : votre structure est maintenant construite dans votre monde.
